# Zyron Downloader
Windows GUI application for downloading youtube videos.

# Downloading application
* Current version of application is v1.0.0
* Go to Release / v1.0.0 path & download all 3 parts of zip
* Extract the files to get Zyron Downloader 1.0.0.exe

# Running application
* Just click Zyron Downloader 1.0.0.exe file
* Copy any youtube link and click on paste link
* It would display list of all available streams with resolution & quality details
* By default the best quality is selected, if required select the quality & format of your choice
* Click download, the selected video would be downloaded

# Note: 
* This GUI internally uses yt-dlp & ffmpeg commandline tools
